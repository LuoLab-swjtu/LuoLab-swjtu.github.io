# LuoLab-swjtu.github.io

Official website for Luo Lab at Southwest Jiaotong University.

## Tech Stack

- React
- Vite
- GitHub Pages

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

This repository deploys to GitHub Pages through GitHub Actions. Pushes to the `main` branch build the Vite app and publish the generated `dist` directory.
